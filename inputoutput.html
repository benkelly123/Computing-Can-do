Inputs and Outputs

In python inputs are delt with with the input() statement. Output is delt with by the print() statement.

# Example of input and output in python
MyInput = str(input("Enter your name: "))
print("Hello " + MyInput)

String handling

# Adding strings together
string1 = 'ben'
string2 = 'computer'
string3 = string1 + string2
print(string3)
>>>'bencomputer'
# Splitting strings
string1 = 'this is python'
array = string1.split(' ')
print(array)
>>>('this', 'is', 'python')

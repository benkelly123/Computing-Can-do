Lists and Tuples and Dictionaries

Lists

In python lists are used instead of arrays. The difference between these is that lists can contain elements of different data types, whereas an array can only contain elements of the same data type.

# Lists are defined with:
list1 = []
# You can add to a list by:
list1 = list1.append("data")
list1 = list1.append("other data")
# To access an element in an array:
print(list1[0])
>>> "data" # In python the first element will always be at the index 0
# print all of the elements in an array:
print(list1)
>>> ["data", "other data"]

Tuples

Tuples are the same as lists in python, with one difference, tuples cannot be changed in the code, they are constant.

# To create a tuple:
tuple1 = (45, 65, 778)
tuple2 = (67, ) # To create a single element you still need to have a comma after the first value
# Access values in a tuple
print(tuple1[0]) # This will output the first element in the tuple.
>>> 45
# Combining tuples
tuple3 = tuple1 + tuple2
print(tuple3)
>>> (45, 65, 778, 45) # tuple2 has been added to the end of tuple1

Dictionaries

A dictionary can be used to associate different values with eachother. This is done by using a key. The key and the value that it associates with are seperated vy a :, different elements in the dictionary are seperated by a comma.

# To create a dictionary:
dict1 = {}
# To add items to a dictionary:
dict1["name"] = "James"
# To output a dictionary value:
print(dict1["name"])
>>> "James"
# To update an already existing value in a dictionary:
dict1["name"] = "Alex"
print(dict1["name"])
>>> "Alex"
# Remove dictionary elements:
del dict1["name"] # Deletes the element relating to the key "name"
dict1.clear() # Clears all of the elements in the dictionary
del dict1 # Deletes the whole dictionary

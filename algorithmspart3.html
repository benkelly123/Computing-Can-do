Addition

In python addition can simply be acheived with the sign +, however it is important to know how this works.

# Addition example
def inc(x):
# Returns x + 1
return x+1
def dec(x):
#returns x - 1
return x-1
def peano(x, y):
#Takes two integers and adds them using peano addition
if y == 0:
return x
else:
return peano(dec(x), inc(y))
#main
if __name__ == "__main__":
print("Peano addition")
a = int(input("First Number: "))
b = int(input("Second Number: "))
print(peano(a, b))

This program uses the peano method for addition. This method utilises recursion.

a+0=a
a+S(b)=S(a+b)

The program works by taking 1 from x and adding 1 to y until x is equal to 0. This will mean that the two integers have been added together.

Subtraction

The subtraction algorithm works in the same way as the addittion algorithm. This means we can compact the code.

# Subtraction example
def peano(x, y):
#Takes two integers and adds them using peano addition
if y == 0:
# Return the value added
return x
else:
# Recursion of the peano function
return peano(x-1, y-1)
#main
if __name__ == "__main__":
print("Peano addition")
a = int(input("First Number: "))
b = int(input("Second Number: "))
print(peano(a, b))

The program works by taking 1 away from both numbers each time.

a−0=a
0−S(b)=0
S(a)−S(b)=a−b

Division

Division is effectively repeated subtraction.

# Division Example
def peano_minus(x, y):
#Takes two integers and adds them using peano addition
if y == 0:
# Return the value added
return x
else:
# Recursion of the peano function
return peano_minus(x-1, y-1)
def divide(x, y, result):
# Repeatedly subtracts y from x
# 'result' stores the number of subtractions made
# Check to see if we are finished
if x < y:
if x == 0:
# Return the answer in a list
return [result, 0]
else:
# When there is a remainder
return [result, 0]
else:
# Call the subtraction algorithm
x = peano_minus(x, y)
# Recursively call 'divide', with result incremented by one
return divide(x, y, result+1)
#main
if __name__ == "__main__":
print("Peano division.")
a = int(input("First Number: "))
b = int(input("Second Number: "))
ans = divide(a, b, 0)
if ans[1] == 0:
print(ans[0])
else:
print(ans[0], " Remainder ", ans[1])

Part 1 - Square and Cubed Numbers

Example Code for Squaring a Number

# square.py - simple squaring program
def square(x):
# Takes a number and returns its square
return x**2
# main
print(square(int(input("Number to square: "))))

Cubeing a number

The same code as before can be used, with the simple addition of putting the number to the power of 3, instead of 2.

# cubeing program
def square(x):
# Takes a number and returns its cube
return x**3
# main
print(square(int(input("Number to cube: "))))

Program example

This program raises a number to the power of any whole number.

# power.py - demonstrates raising numbers to integer powers by successive squaring
def power(x, y):
# Takes a real numebr x and an integer y and returns x^y
z = 1
while y != 0:
if y % 2 == 0:
x *= x
y //= 2
else:
x *= x
y -= 1
return z
#main
print(" exponetation by successive squaring ")
print("This program will raise x to the power of y")
print("as long as y is a whole number!")
a = float(input("x: "))
b = int(input("y: "))
print("Result: ", power(a, b))
